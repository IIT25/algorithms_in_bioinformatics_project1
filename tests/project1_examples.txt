
Test cases for global alignment usign linear gap cost
-----------------------------------------------------

Consider the below score matrix M for dna sequences:

   A  C  G  T
A  0  5  2  5
C  5  0  5  2
G  2  5  0  5
T  5  2  5  0


Case 1
------

>seq1
acgtgtcaacgt 

>seq2
acgtcgtagcta 

Using score matrix M and linear gap cost g(k)=5*k, the score of an optimal alignment is:

22

and there are 2 optimal alignments:

acgt-gtcaacgt-
acgtcgt-agc-ta

acgt-gtcaacgt
acgtcgt-agcta

Case 2:
-------

>seq1
aataat

>seq2
aagg

Using score matrix M and linear gap cost g(k)=5*k, the score of an optimal alignment is:

14

and an optimal alignment is:

aataat
aa-gg-
